# Studentų galutinio balo skaičiuoklė (v0.1)

Programa nuskaito studentų vardą, pavardę, namų darbų pažymius (ND) ir egzamino pažymį (Egz), tada išveda **du galutinius balus**:
- **Galutinis (Vid.)** – naudojant ND vidurkį
- **Galutinis (Med.)** – naudojant ND medianą

Formulė: `Galutinis = 0.4 * ND + 0.6 * Egz`

## Failai
- `main_array.cpp` – versija su **C masyvais** (be `std::vector`)
- `main_vector.cpp` – versija su **std::vector**
